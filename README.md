# Rodrigo mdBook Portfolio Starter

This is a starter project for a professional architecture portfolio using **mdBook**, **Markdown**, **Mermaid**, and **GitHub Pages**.

## What's included

- Structured navigation for your architecture portfolio
- Ready-to-edit content pages
- Mermaid diagrams already enabled
- GitHub Actions workflow for automatic deployment
- A deployment guide for beginners

## Main files

- `book.toml` : mdBook configuration
- `src/` : site content
- `src/SUMMARY.md` : left-side navigation
- `.github/workflows/deploy.yml` : automatic deployment to GitHub Pages

## Before publishing

Update:

- `YOUR_GITHUB_USERNAME` in `book.toml`
- LinkedIn and email in `src/contact.md`
- any content you want to personalize further
