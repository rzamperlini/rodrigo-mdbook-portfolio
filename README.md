# Rodrigo's mdBook Portfolio

This is a project for a professional architecture portfolio using **mdBook**, **Markdown**, **Mermaid**, and **GitHub Pages**.

## What's included

- Structured navigation for my architecture portfolio
- Ready-to-edit content pages
- Mermaid diagrams enabled
- GitHub Actions workflow for automatic deployment
- A deployment guide

## Main files

- `book.toml` : mdBook configuration
- `src/` : site content
- `src/SUMMARY.md` : left-side navigation
- `.github/workflows/deploy.yml` : automatic deployment to GitHub Pages
